# Demo Seed Data

This directory contains synthetic seed data intended for public portfolio demos.

Principles:
- Built from the *distribution* of `/home/kihun/ilowa/ai_modeling/data/job_seed_seniorforum_100.json` (not copied).
- No PII or source material: `address`, `contact_phone`, `source_url` are removed.
- Locations are coarse (`location_label`) at gu/dong granularity only.
- Coordinates are synthetic (jittered within the allowed bounding box + ~6km radius).
- Deterministic generation (seed=42) for reproducibility.

Fields in `demo_jobs_50.json`:
- `job_id`, `title`, `description`, `category`, `place`
- `work_days`, `start_time`, `end_time`, `participants`, `hourly_wage`
- `lat`, `lng`, `location_label`
- `client`, `source_type`, `source_note`

`demo_jobs_50_with_embeddings.csv` (generated locally, not tracked):
- CSV schema aligned with `new_work_with_embeddings.csv`
- `embedding` column uses deterministic pseudo-vectors (seed=42) generated by
  `ai_modeling/scripts/build_demo_csv_from_json.py`
